Padding

Use the padding utility to change an element's padding. The [MULTIPLE] is a number 0-7.

Format: ds-u-padding-[x|y|bottom|left|right|top]--[MULTIPLE]

sba-u-padding--0

sba-u-padding--1

sba-u-padding--2

sba-u-padding--3

sba-u-padding--4

sba-u-padding--5

sba-u-padding--6

sba-u-padding--7

Responsive padding

The padding utility also supports breakpoint prefixes.

Redesign of Application Overview

The goals of the redesign are the following:

  • Provide application in a more digestible and scannable manner
  • Remove unnecessary data points to reduce noise on page
  • Place Ad Hoc Reviews in line with the application
  • Aside from bringing Ad Hocs in line, the general functionality is about the same as the existing overview, but with a visual refresh

This is only a design refresh on the cards in the application overview.

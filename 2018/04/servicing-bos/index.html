Servicing BOS

This prototype will include a method to add and change a servicing BOS. In order for the UI to support this new information, a new header needs to be supplied.

Important!

This prototype is limited. The user should change the district office to Louisiana, and then select a servicing BOS from the Alabama district office.

Some rules

  • Servicing BOS is set to "None" by default
  • The list of servicing BOS that appears by default would be the BOS's that correpsond with the firm's current district office.

Dev note

  • All lists of BOS's for all field offices are included on the page, but hidden with a hidden attribute.
  • The list filters by adding a hidden attirbute to all sets of BOS's and then removing the hidden attribute for only the corresponding field office selected in the drop down.

This is not set in stone. AJAX would also be acceptable for loading the BOS lists on the page.

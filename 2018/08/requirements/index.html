Adding documents

This protype is functional. In this prototype, you will be able to:

  • Attach a document ('type' and a file are required)
  • Remove a document

This prototype will show both a null state and a state displaying a single document.

Note for developers: Please remove any instance of data-filter-* and data-print from the HTML when converting to slim templates. These data attributes pertain only to the prototyping engine.

Also, documents on this form are not restricted to PDF. If they must be a PDF, please be sure to add that in production.

The states of this page will require parameters to function properly. The initial parameter is docs=false. Once a document is added, the url parameter will be become docs=true

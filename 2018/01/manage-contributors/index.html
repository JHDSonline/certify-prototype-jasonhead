Managing contributors

This will provide a much more informed process for adding contributors to an 8a application. The approach is meant to show how this can be accomplished in our existing system and when contributors can be added at the firm level.

  • The application is an initial application
  • Only the Reassign case workflow is functional
  • Any discrepancies in the application overview should be ignored
  • No other pages, such as messages, documents, etc. are available
  • If the Supervisor assigns the case to another District Office, the question to assign a Servicing BOS is skipped

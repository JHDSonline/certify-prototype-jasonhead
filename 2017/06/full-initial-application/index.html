Notes for Certify Team

The Task Bar (AKA Fun Bar)

  • The task bar will be sticky when the user scrolls to it.
  • The options in the fun bar will be different depending on the status of the application and the role of the user.

Ad Hoc Reviews

Ad Hoc Reviews are only available to applications that are in processing, so you'll need to change the status before you see the link.

To initiate an ad hoc review, either select "Request info from firm" from the task panel or go to the additional info section at the bottom of the application overview.

Next, the analyst will use a form to request info.

  • All fields are required
  • At least one checkbox must be checked.
  • The cancel button will return user to previous page

After the request is submitted, a new card will be created. New cards will be created for each ad hoc request, with the latest card appearing first.

Firm Withdraws a case

A firm may choose to withdraw a case. When that happens, an alert message will appear on the top of the application view in the masthead. To view the alert, you must apply this URL parameter: withdrawal=true
